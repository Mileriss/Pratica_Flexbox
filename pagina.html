Flexbox

Flexbox permite criar layouts com design e estrutura responsivo e sem necessidade de uma configuração adicional de geometria.

Tipos de Layouts

  • Block:
    • Utilizado para seções de uma página web
  • Inline:
    • Utilizado para textos
  • Table:
    • Para dados de uma tabela bi-dimensional
  • Positioned:
    • Para posições explicitas de um elemento

CONTAINER

Flex container torna os elementos flexiveis dentro do layout e disponibiliza algumas propriedades para sua manipulação.
